COMO EXECUTAR O PROGRAMA:

Metodo 1 (recomendado):
- Ir até o cmd e navegar ate o arquivo, exemplo, use o comando:
--> cd ../EntregaA3/src

- Depois de estar na pasta do arquivo, execute o programa usando o comando
--> python Main.py

Metodo 2:
- Instalar uma IDE e executar o programa pelo pelo proprio software, recomenda-se o "vscode" caso opte por executar o arquivo dessa maneira
- Execute o arquivo "Main.py" pelo botao de executar da IDE, no vscode é uma seta "Run" que se localiza no canto superior direito
